# CloudSage AI

CloudSage AI is an AI-powered DevOps monitoring and log analysis platform built using a production-grade 3-tier architecture on AWS.

## Features

- DevOps monitoring dashboard
- AI-powered log analysis
- Infrastructure monitoring
- DevOps troubleshooting chatbot
- CI/CD automation
- Cloud deployment using Terraform

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript / React

Backend:
- Node.js
- Express.js

AI Engine:
- Python
- OpenAI API

Infrastructure:
- AWS (EC2, ALB, RDS, VPC)
- Terraform
- Docker
- GitHub Actions
